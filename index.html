Tạo project Hello World

B1: Tạo thư mục trong File Explorer viết liền không dấu: Bai1
B2: Trong thư mục Bai1 → chuột phải → Open with code
B3: Chọn New File → đặt tên file: HelloWorld.cpp (đuôi file c++ là .cpp)
B4: Dán đoạn code vào:
C++
Copier
#include <iostream> using namespace std; int main() { cout << "Hello World!!"; return 0; }
